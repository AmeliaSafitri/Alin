Pagerank#

PageRank bekerja dengan menghitung jumlah dan kualitas tautan ke suatu halaman untuk menentukan perkiraan kasar seberapa penting situs web tersebut. Asumsi yang mendasarinya adalah bahwa situs web yang lebih penting cenderung menerima lebih banyak tautan dari situs web lainnya

import numpy as np

A = np.array([[0, 0, 1, 1/2],
              [1/3, 0, 0, 0],
              [1/3, 1/2, 0, 1/2],
              [1/3, 1/2, 0, 0]])

v = np.array([[0.25], [0.25], [0.25], [0.25]])

previous_result = v
tolerance = 0.01  # Ubah nilai toleransi menjadi 0.01

for i in range(1, 100):  # Batasi jumlah iterasi untuk mencegah perulangan tak terbatas
    result = A @ previous_result
    print(f"(A^{i})v = \n", result)

    # Cek kondisi berhenti
    if np.all(np.abs(result - previous_result) <= tolerance):
        if np.sqrt(np.sum((result - previous_result) ** 2)) < tolerance:
          print(f"\nPerkalian matriks konvergen pada iterasi ke-{i}.")
          break

    previous_result = result

import numpy as np

A = np.array([[0, 1/2, 1, 1/3, 0],
              [0, 0, 0, 1/3, 1/3],
              [0, 1/2, 0, 0, 1/3],
              [1, 0, 0, 0, 1/3],
              [0, 0, 0, 1/3, 0]])

v = np.array([[0.20], [0.20], [0.20], [0.20], [0.20]])

previous_result = v
tolerance = 0.01

# List untuk menyimpan hasil PageRank dari setiap iterasi
pagerank_results = []

for i in range(1, 100):
    result = A @ previous_result
    pagerank_results.append(result.flatten())

    print(f"Iterasi ke-{i}:")
    for node, value in enumerate(result):
        print(f"Node {node+1}: {value[0]}")
    print()
    if np.all(np.abs(result - previous_result) <= tolerance):
        if np.sqrt(np.sum((result - previous_result) ** 2)) < tolerance:
            print(f"\nPerkalian matriks konvergen pada iterasi ke-{i}.")
            break

    previous_result = result

# Setelah loop selesai, mengurutkan hasil PageRank dari tertinggi ke terendah
final_pageranks = np.array(pagerank_results[-1])  # Mengambil hasil dari iterasi terakhir
sorted_indices = np.argsort(final_pageranks)[::-1]  # Mengurutkan indeks dari nilai tertinggi ke terendah

print("\nHasil PageRank setelah konvergensi (dari tertinggi ke terendah):")
for rank, idx in enumerate(sorted_indices):
    print(f"Rank {rank+1}: Node {idx+1} dengan nilai {final_pageranks[idx]}")

Algoritma#

Berikut adalah langkah-langkah dasar dari algoritma PageRank:

  1. Model Surfer Acak (Random Surfer Model): Algoritma ini mengasumsikan bahwa pengguna internet (surfer) secara acak mengklik tautan dari satu halaman ke halaman lain. Probabilitas halaman untuk dikunjungi mencerminkan pentingnya halaman tersebut.

  2. Distribusi Tautan: Setiap halaman membagi pentingnya (ranking) secara merata ke semua halaman yang ditautkannya. Jika sebuah halaman memiliki banyak tautan keluar, maka setiap tautan akan mendapatkan bagian penting yang lebih kecil.

  3. Faktor Damping: Untuk mencegah semua ranking terkonsentrasi pada beberapa halaman, digunakan faktor damping (biasanya sekitar 0.85). Ini mengasumsikan bahwa surfer tidak hanya mengklik tautan, tetapi juga bisa melompat ke halaman acak dengan probabilitas tertentu (1 - d).

  4. Perhitungan Iteratif: PageRank dihitung secara iteratif, di mana setiap iterasi memperbarui ranking berdasarkan ranking yang diterima dari halaman lain. Proses ini diulang hingga ranking stabil dan tidak berubah signifikan antar iterasi.

Untuk menentukan pagerank sebuah halaman web, Google menggunakan algoritma sebagai berikut :

   PR(A) = (1-d) / N + d ( ( PR(T1) / C(T1) ) + … + ( PR(Tn) / C(Tn) ) )

Keterangan :

PR(A) adalah Pagerank halaman A

PR(T1) adalah Pagerank halaman T1 yang mengacu ke halaman A (yang dimaksud adalah pagerank dari backlink yang mengarah ke situs A)

C(T1) adalah jumlah link keluar (outbound link) pada halaman T1 (yang dimaksud adalah jumlah link keluar dari backlink yang mengarah ke situs A)

d adalah damping factor yang bisa diberi antara 0 dan 1

N adalah jumlah keseluruhan halaman web yang terindeks oleh Google

Komputasi untuk menghitung pagerank (Gunakan error toleransi)#

  1. Toleransi kesalahan mengacu pada sejauh mana algoritma dapat mengatasi ketidakpastian dalam perhitungan.

  2. Dalam konteks PageRank, toleransi kesalahan dapat diterapkan dengan membatasi jumlah iterasi atau menghentikan iterasi ketika perubahan skor PageRank sudah cukup kecil.

  3. Dengan memperkenalkan toleransi kesalahan, kita dapat mengurangi waktu komputasi dan memori yang dibutuhkan.
