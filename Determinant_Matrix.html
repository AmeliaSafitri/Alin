DETERMINANT MATRIX#

Determinan matriks adalah nilai skalar yang bisa dihitung dari elemen-elemen sebuah matriks persegi. Matriks persegi adalah matriks yang jumlah baris dan kolomnya sama, seperti 2x2 atau 3x3

Determinan matriks ordo(2 x 2) $\( \begin{equation*} A= \begin{bmatrix} a&b\\ c&d \end{bmatrix} \end{equation*} \)\( \)\( \begin{equation*} det (A) = ad - bc \end{equation*} \)$

Determinan matriks ordo (3 x 3) $\( \begin{equation*} A= \begin{bmatrix} p&q&r\\ s&t&u\\ v&w&x\ \end{bmatrix} \end{equation*} \)\( \)\( \begin{equation*} det (A) =ptx + quv + rst - vtr - wuq - xsq \end{equation*} \)$

Syarat-syarat Determinan Matriks#

Determinan Matriks Persegi yaitu determinan yang hanya bisa dihitung untuk matriks persegi, yaitu matriks dengan jumlah baris dan kolom yang sama

Fungsi Determinan Matriks#

1. Determinan dari matriks identitas akan mengkasilkan 1#

import numpy as np

determinan = np.linalg.det(A)

4.#

\[\begin{split} \begin{equation*} (A)= \begin{bmatrix} a&b\\ c&d \end{bmatrix} \end{equation*} \end{split}\]

jika matriks ( A ) dikalikan dengan skalar ( x ), lalu menghasilkan matriks ( B ), maka determinan matriks ( B ), akan sama dengan ( x^n ) dikalikan dengan ( det(A) ), di mana ( n ) adalah ordo matriks. Ini berarti jika ( x ) dikuadratkan dan kemudian dikalikan dengan determinan matriks ( A ), hasilnya akan memberikan determinan matriks ( B ), asalkan ( n ) adalah ordo kuadrat dari matriks tersebut.

\[\begin{split} \begin{equation*} (B)= \begin{bmatrix} X*a&X*b\\ X*c&X*d \end{bmatrix} \end{equation*} \end{split}\]

Matriks tersebut memiliki ordo (2x2). Kemudian mencari determinan matriks A dan B. maka akan menghasilkan sebuah fungsi yaitu

\[ \begin{equation*} det(B) = X^n * det(A) \end{equation*} \]
import numpy as np
A =np.array  ([[3,4], [2,1]])
determinan = np.linalg.det(A)
print (A, "\n")
print("Determinan dari matriks A adalah:", int(determinan))

H = np.array ([[6,8], [4,2]])
determinan_H = np.linalg.det(H)
print (H, "\n")
print("Determinan dari matriks H adalah:", int(determinan_H.round()))

print ('det(B) = X^n * det(A)= 2^2 * (-5)= 4 * (-5)= (-20)')

import numpy as np

def determinan_submatrix(matriks, i, j):
    # Menghapus baris i dan kolom j untuk mendapatkan submatriks
    submatriks = np.delete(np.delete(matriks, i, axis=0), j, axis=1)
    # Menghitung determinan dari submatriks
    return np.linalg.det(submatriks)

def hitung_c_ij(matriks, i, j):
    # Menghitung determinan dari submatriks dan mengalikannya dengan (-1)^(i+j)
    determinan = (-1) ** (i + j) * determinan_submatrix(matriks, i, j)
    return determinan

# Matriks yang diberikan
matriks = np.array([[3, 1, 3, 1, 2],
                    [2, 1, 3, 1, 3],
                    [3, 2, 4, 1, 2],
                    [1, 3, 1, 2, 4],
                    [3, 2, 2, 1, 4]])
print ('Join matriks')
# Menghitung c_ij untuk semua elemen dari baris 1 hingga 5 dan kolom 1 hingga 5
for i in range(len(matriks)):
    for j in range(len(matriks[i])):
        hasil = hitung_c_ij(matriks, i, j)
        print(f"baris {j+1} kolom {i+1}: {hasil}")
